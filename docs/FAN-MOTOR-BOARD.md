## Τυπικό μοτέρ outdoor fan (PSC)

Στα περισσότερα split AC/HP outdoor units, το μοτέρ βεντιλατήρα condenser είναι **PSC** (Permanent Split Capacitor), **single-speed** ή **two-speed**.

### Συχνές χρωματικές (ΗΠΑ / universal · **πάντα διάβασε το diagram στο μοτέρ**)

| Χρώμα | Συνήθης ρόλος |
|-------|----------------|
| **Black** | Line / HIGH speed → contactor ή relay |
| **White** | Common / Neutral |
| **Red** ή **Yellow** | LOW speed (2η ταχύτητα — συχνά **δεν** χρησιμοποιείται) |
| **Brown** | Ένα άκρο πυκνωτή run |
| **Brown/White** | Άλλο άκρο πυκνωτή (ή κοινό με C) |

→ **5 καλώδια = 3 λειτουργικά + 2 για πυκνωτή**  
ή **5 = common + high + low + 2× cap**.

### Τι περιμένει συχνά πλακέτα με **4 αναμονές**

Τυπική σχεδίαση «4 σημεία»:

```
J1  LINE      (black → από contactor/relay)
J2  COMMON    (white → N ή C πυκνωτή)
J3  CAP_A     (brown)
J4  CAP_B     (brown/white)
```

**Ή** (αν ο πυκνωτής είναι **on-board** στην πλακέτα):

```
J1  LINE
J2  COMMON  
J3  (κενό / speed — δεν χρησιμοποιείται)
J4  (κενό)
```

με **1** σημείο cap εσωτερικά — τότε τα 2 brown του μοτέρ πάνε **μαζί** σε μία αναμονή + το άλλο άκρο cap στο C.

---

## Γιατί 5 καλώδια vs 4 αναμονές — πιθανές αιτίες

### 1. Δεύτερη ταχύτητα (πιο συχνό)

Το μοτέρ είναι **2-speed** αλλά η πλακέτα σχεδιάστηκε για **1 speed**:
- Χρησιμοποιείς: black (high) + white (common) + 2× brown (cap)
- **5ο καλώδιο:** red ή yellow (low) → **απομονώνεται** (tape / Wago, not connected)

**Δεν χρειάζεται 5η αναμονή** αν τρέχεις μόνο high.

### 2. Δύο καλώδια πυκνωτή vs μία αναμονή cap

Αν η πλακέτα έχει **ένα** pad «CAP» και εσωτερικό πυκνωτή:
- Τα **2 brown** του μοτέρ μπαίνουν στο **ίδιο** σημείο (ή σε σειρά με το cap της πλακέτας)
- Μετράς **4 εμφανή** καλώδια προς πλακέτα, όχι 5 ξεχωριστές αναμονές

### 3. Γείωση (PE) — 5ο καλώδιο

Αν το 5ο είναι **πράσινο/κίτρινο-πράσινο (PE)**:
- **Δεν** πάει στις 4 αναμονές σήματος
- Πάει σε **επίγειο** σημείο πλαισίου / PE terminal
- Η πλακέτα μπορεί να **έλειπε** ξεχωριστό PE pad (λάθος omission στο schematic)

### 4. Heat pump vs απλό AC (defrost)

Σε **HP**, το fan condenser συχνά περνά από **defrost board** — black **δεν** πάει απευθείας στο contactor όπως σε AC.
- Μπορεί να χρειάζεται **επιπλέον** control line (relay από defrost)
- Αυτό μπορεί να εξηγεί «ένα» επιπλέον καλώδιο στο σύστημα (όχι απαραίτητα 5ο pad στο μοτέρ)

## Checklist — τι να μετρήσεις στο bench

- [ ] **Χρώματα** και **ετικέτα diagram** στο πλαίσιο του μοτέρ (φωτο)
- [ ] **Αντίσταση** καλωδίων (με multimeter, μοτέρ αποσυνδεδεμένο):
  - Common ↔ κάθε speed tap
  - Brown ↔ brown/white (πυκνωτής winding)
- [ ] **Τι λένε οι 4 αναμονές** στην πλακέτα (silkscreen: L, N, C, F, κ.λπ.)
- [ ] Υπάρχει **πυκνωτής on-board**; τι μF;
- [ ] Το 5ο καλώδιο είναι **PE** ή **signal**;
