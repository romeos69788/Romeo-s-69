# Σημείο επανακίνησης — οθόνη Love (`s_scr_love`)

**Ημερομηνία:** 2026-04-13  
**Κώδικας:** `thermostat-ui-demo/src/main.cpp` (LVGL χειροκίνητη διάταξη, όχι SquareLine export για αυτή την οθόνη).

## Σκοπός

Μετάβαση από `ui_Romeos1`: **δεξί βελάκι** όταν το ρολόι **δεν** είναι σε επεξεργασία → `lv_scr_load_anim(s_scr_love, MOVE_LEFT, …)`.  
Επιστροφή: **← Main** στη γραμμή MENU (ή αντίστοιχο handler), ή από την τρίτη οθόνη **Back**.

## Διάταξη

- **7 γραμμές** (`grid` × `r`): αριστερά στήλη ρολών (`lc`), δεξιά (`rc`) — `flex_grow` 1:1, `pad_column(r) = 24`.
- **Ύψος γραμμής:** `k_shutter_row_outer_h = k_shutter_bar_h + 10` (= **66 px**) — σταθερό ώστε να χωράει το 7× στο **480 px** ύψος· τα στρογγυλά κουμπιά **65×65** μπορεί να «βγαίνουν» οπτικά (`OVERFLOW_VISIBLE` όπου χρειάζεται).
- **Μπάρα κειμένου:** `k_shutter_bar_h = 56`, στυλ `squareline_uiverse_shutter_bar_style`, `flex_grow` στο `strip`· **πλάτος μπάρας** = πλάτος στήλης − **65 − 8 − 65 − 8** (κύκλοι + `pad_column` strip **8**) — σε **800 px** πλάτος και pad οθόνης **10+10** περίπου **~232 px** (όχι ~400· γεωμετρικό όριο με δύο στήλες και κύκλα 65).

## Γραμματοσειρές (Montserrat)

| Στοιχείο | Μέγεθος |
|----------|---------|
| Κείμενο στις μακρόστερες μπάρες (+ ονόματα φώτων, MENU) | **28** |
| ON / OFF στα στρογγυλά φώτων | **22** |
| Σύμβολα ▲/▼ ρολών, βέλη πλοήγησης | **28** |

Labels μπάρας: `LV_LABEL_LONG_DOT` + **`max_height` = ύψος μίας γραμμής** (χωρίς wrap σε δύο σειρές)· για μακριά ονόματα χρησιμοποιούνται **σύντομα strings** (π.χ. `RM-L` / `RM-R`, `1/FL` / `2/FL` / `-1/FL`).

## Σταθερές μετατόπισης (οπτικό κενό)

- `k_love_left_bar_nudge_px = -24` — `translate_x` στη **μπάρα** αριστερά.
- `k_love_right_btn_pair_nudge_px = 24` — `translate_x` και στα **δύο** στρογγυλά δεξιά (χωρίς ενδιάμεσο `pair` flex).

## Ονόματα (arrays)

- `k_love_left_names[7]` — αριστερά ρολά (Latin· το LVGL Montserrat εδώ χωρίς ελληνικά subset).
- `k_love_right_shutter_name` — πρώτη δεξιά γραμμή ρολού.
- `k_love_right_light_names[5]` — φώτα + EXIT (indices shutter rows στο `s_shutter_rows` όπως στο `main.cpp`).

## Τρίτη οθόνη

- `s_scr_love_next` — δοκιμαστική / «επόμενη» οθόνη από **→** στη γραμμή MENU.

## Build

- PlatformIO env: **`BOARD_VIEWE_UEDX80480070E_WB_A`** (`platformio.ini`).

## Επόμενη φορά (υπόμνημα)

- Σύνδεση κουμπιών ρολών/φώτων με **πραγματική** λογική (MB/UDP) αν χρειάζεται· σήμερα το UI είναι κυρίως **demo κατάστασης** (`s_shutter_pos` κ.λπ.).
- Fine-tune copy ονομάτων (`RM-L` κ.λπ.) αν ο χρήστης θέλει άλλη ονοματοδοσία.
- Χωρίς αλλαγή ανάλυσης panel, **~400 px** μπάρα **και στις δύο** στήλες δεν χωράει μαζί με κύκλα **65** — χρειάζεται συμβιβασμός (μικρότερα κύκλα, ασύμμετρες στήλες, ή άλλη διάταξη).
