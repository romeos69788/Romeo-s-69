# Μικρόφωνο I2S — ίδιο εξάρτημα ×2 (μητρική + θερμοστάτης 7″)

**Σκοπός:** πριν την παραγγελία PCB να υπάρχει κοινή επιλογή εξαρτήματος και σύνδεσης ώστε να παραγγείλεις **δύο ίδια κομμάτια** — ένα για τη **μητρική** και ένα για την **οθόνη θερμοστάτη 7″** (ίδια λογική firmware/driver, διαφορετικό ESP/chip ανά πλακέτα).

---

## 1. Προτεινόμενο εξάρτημα

| Επιλογή | Περιγραφή |
|--------|------------|
| **Πρωτεύουσα** | Έτοιμο **breakout module I2S** με **InvenSense INMP441** (στην αγορά συχνά ως «**INMP441 I2S microphone module**» / σειρά τύπου GY-INMP441 **ή** στρογγυλό PCB ~**14 mm Ø** με pads **SCK, WS, SD, L/R, VDD, GND** και θύρα στο κάτω μέρος — ίδια ηλεκτρική λογική). |
| IC (για αναφορά BOM/LCSC) | **INMP441ACEZ7** — στα modules είναι ήδη συγκολλημένο· δεν χρειάζεται να το αγοράσεις χωριστά αν παίρνεις έτοιμη πλακέτα module. |
| **Εναλλακτικό** | Breakout με **ICS-43434** (ίδια ιδέα: I2S, 3V3, τρεις γραμμές ρολογιού/δεδομένων). |

**Γιατί I2S και όχι απλό εκλεκτέτ:** ψηφιακή έξοδος, καλύτερο SNR για μελλοντική φωνητική διεπαφή / wake-word, λιγότερο ευαίσθητο σε θόρυβο μπουκ από αναλογικό + ADC του ESP.

---

## 2. Σύνδεση προς ESP32 (κοινή για τις δύο συσκευές)

Τα περισσότερα modules INMP441 φέρουν 6 ακροδέκτες (ή αντίστοιχα pads):

| Ακίδα module (τυπικά) | Σύνδεση |
|------------------------|---------|
| **VDD** / 3V3 | **3V3** λογικής (όχι 5 V) |
| **GND** | **GND** λογικής |
| **WS** / LRCLK | Γραμμή **I2S WS** (word select) του ESP |
| **SCK** / BCLK | Γραμμή **I2S bit clock** του ESP *(πολλά modules γράφουν **SCK**)* |
| **SD** / DIN προς ESP | **Δεδομένα** από το μικρόφωνο → είσοδος **I2S DIN** του ESP |
| **L/R** | Σύνδεση σε **GND ή 3V3** ανάλογα με το module για **κανάλι αριστερό/δεξιό** (διάβασε το silkscreen του συγκεκριμένου module — συχνά ένα κανάλι αρκεί). |

**Σημαντικό:** κοινό **GND** με τον ESP· **κοντό ίχνος** στο PCB· μακριά από κουμπωμένα **ρελέ / 220 V** (θωράκιση/διάταξη).

---

## 3. Δύο τεμάχια στην παραγγελία

- Παραγγέλνεις **2× το ίδιο module** (ίδιος κατασκευαστής/breakout αν γίνεται) ώστε:
  - ίδια πολικότητα L/R και σήμανση pins,
  - ίδια συμπεριφορά στο firmware (ίδια ρύθμιση I2S),
  - λιγότερα λάθη στην εγκατάσταση.

Η **μητρική** (ESP32 **DevKit 38-pin**) και ο **θερμοστάτης 7″** (συνήθως **ESP32‑S3** + RGB panel) έχουν **διαφορετικό pinout** — οι **αριθμοί GPIO δεν είναι οι ίδιοι**, αλλά το **καλώδιο λειτουργικά** (3V3, GND, WS, BCLK, DIN) είναι **ίδιο**.

---

## 4. PCB μητρικής — τι να προσθέσεις πριν το gerber

1. **Κλέμα / header** π.χ. **1×6 pin 2.54 mm** (ή JST-SH αν προτιμάς) με ετικέτα **`MIC_I2S`** στο silkscreen.
2. Σύνδεση στα **GPIO** που θα επιλεγούν στο **EasyEDA** για **I2S master RX** (3 γραμμές + τροφοδοσία).
3. **Προσοχή:** το τρέχον `motherboard-fw/include/pin_map.h` είναι **ήδη πλήρες** σε χρήσεις — για I2S χρειάζεσαι **τρία ελεύθερα GPIO** ή **επανεκχώρηση** / επιπλέον έκθεση pins από το DevKit στο schematic. Αυτό είναι **υποχρεωτικό βήμα πριν το κλείδωμα** του routing (δες §5).
4. **Δεν** αφαιρείται η κλέμα **LCD 2004** ούτε η **U4** — μένουν όπως συμφωνήθηκε για την παραγγελία· το μικρόφωνο είναι **πρόσθετο**.

## 5. Κλείδωμα GPIO (μητρική) — **κλειδωμένο 2026-05-08**

| Net | GPIO | ESP header | PCB |
|-----|------|------------|-----|
| `MIC_I2S_BCLK` | **2** | H1 pin **5** | → J_MIC pin **3** (SCK) |
| `MIC_I2S_WS` | **12** | H2 pin **13** | → J_MIC pin **4** (WS) + **10kΩ → GND** |
| `MIC_I2S_SD` | **0** | H1 pin **6** | → J_MIC pin **5** (SD) + **10kΩ → 3V3** (boot) |

Πλήρης checklist κλέμας: **`MIC_PCB_LOCK_2026-05-08.md`**. Firmware: `motherboard-fw/include/pin_map.h` (`PIN_MIC_I2S_*`).

---

## 6. Θερμοστάτης 7″

- Ίδιο module INMP441· σύνδεση στα **διαθέσιμα I2S pins** του SoC της πλακέτας οθόνης (ESP32‑S3) σύμφωνα με το manual/vendor της Viewe / board JSON του `romeos-display-v4`.
- Τοποθέτηση: **άλλο κουτί / χώρος χρήστη** — σχεδίασε **μικρή οπή ή προστατευτικό πλέγμα** για ακουστική και λιγότερο θόρυβο ρεύματος αν η τροφοδοσία είναι κοντά.

---

## 7. Firmware

- Μητρική και θερμοστάτης: **`driver/I2S`** ή Arduino **`I2S.h`** — ρύθμιση ως **receiver**, τυπικά **16-bit**, sample rate π.χ. **16 kHz** ή **48 kHz** ανά εφαρμογή.
- Η ενεργοποίηση κώδικα μπορεί να έρθει **μετά** την πρώτη συναρμολόγηση — το PCB πρέπει μόνο να **φέρει τα νετάσματα** σωστά.
