# Σημειώσεις συζήτησης (όχι υποχρεωτική υλοποίηση ακόμα)

Αρχείο για να **επιστρέφουμε** σε αποφάσεις/ιδέες χωρίς να χαθούν στο chat.

## 1. Φωνητικές εντολές (μελλοντικά)

### Σενάριο χρήσης

- Σε κάθε οθόνη με εντολές (π.χ. **φώτα**, **θέρμανση**) να υπάρχει κουμπί με **εικονίδιο μικροφώνου**.
- **Push‑to‑talk:** πάτημα → σύντομη εγγραφή → αναγνώριση → εκτέλεση της **ίδιας** λογικής με τα χειροκίνητα **on/off** (π.χ. «εξωτερικά φώτα off»).
- Τα **χειροκίνητα κουμπιά παραμένουν πάντα**· η φωνή είναι **προαιρετική**.

### Τεχνικά (σύνοψη)

- **Τοπικά στο ESP32‑S3** με έτοιμα σετ τύπου ESP‑SR/MultiNet: κυρίως προσανατολισμός σε **αγγλικά/κινέζικα** για plug‑and‑play λέξεις‑εντολές — **όχι** εύκολο «μαγικό» ελληνικό λεξιλόγιο χωρίς επιπλέον δουλειά.
- **Ελληνικά για όλη την οικογένεια:** συνήθως πιο ρεαλιστικό με **STT στο σύννεφο** (Wi‑Fi) ή με **εκπαιδευμένο τοπικό μοντέλο** για λίγες σταθερές φράσεις.
- **Hardware μικροφώνου (όταν φτάσει η υλοποίηση):** προτιμάται ψηφιακό **I2S MEMS** breakout (π.χ. οικογένεια INMP441 κ.λπ.), ανάλογα με διαθέσιμα GPIO στο hardware της οθόνης.

### Setpoint / οθόνη (ήδη διορθωμένο στο firmware)

- Μετά από αλλαγή setpoint από τον χρήστη, η οθόνη **δεν** επανέρχεται στο παλιό `setpoint_c_x10` της μητρικής επειδή «έληξε χρονόμετρο»· συγχρονίζεται ξανά με UDP **μόνο όταν** το πακέτο της μητρικής φέρει την **ίδια** ακέραια τιμή °C (`romeos_v4_setpoint_ui.cpp`).
