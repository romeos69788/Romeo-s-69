# Συζήτηση 2026-04-26 — UI SquareLine, ρολόι, δεύτερες οθόνες, εικόνα σπιτιού

Σύνοψη για **συνέχεια αύριο** (χωρίς αλλαγές κώδικα από αυτή τη συζήτηση — μόνο σχεδιασμός).

## SquareLine Studio — διαφανή κουμπιά

- Σε **κάθε** περιοχή που θέλουμε tap (αριστερή στήλη εικονιδίων, **+/−**, βελάκια ρολογιού, κείμενο ώρας/λεπτών/AM-PM, **MONDAY** / ημερομηνία): **`Button`** με **background opacity 0%**, **Clickable**, προαιρετικά **extended click area**.
- **Ρολόι:** ίδια λογική με το παλιό project — βελάκια / tap σε **ώρα**, **λεπτά**, **AM/PM**, **ημέρα**, **ημερομηνία** για ρύθμιση όταν χρειάζεται (π.χ. μετά από διακοπή ρεύματος).
- **Αριστερή στήλη:** tap π.χ. **LIGHTS** → φόρτωση **δεύτερης οθόνης** μόνο με διακόπτες φωτισμού (εσωτερικά/εξωτερικά). Όμοια **HEAT PUMP** → οθόνα αντλίας (θερμοκρασίες, ώρες λειτουργίας, on/off, φάση κ.λπ.).
- **ALARM:** ίδιο σκεπτικό με πριν — **αναβόσβηση** κόκκινη ολόκληρη οθόνη, **tap στο εικονίδιο alarm** → διακοπή.

## Δεξιά πλευρά — φωτό σπιτιού, εξωτερική T/RH

- **Δεδομένα:** αισθητήρες είτε **μητρική → UDP → οθόνη** είτε **απευθείας στην οθόνη** — απόφαση όταν κλειδώσει wiring· και οι δύο προσεγγίσεις έγκυρες.
- **Lux (φως):** συχνά **στην οθόνη** (ίδια «θέα» με την εικόνα).

## Εικόνα σπιτιού — «μέρα / νύχτα» χωρίς 12-frame video

- **6 εικόνες** (3 ημέρας: ξημέρωμα, μέρα, απόγευμα· 3 νύχτας: σκοτεινιά, βαθιά νύχτα, προ-ξημέρωμα) — **επαρκής** ποιότητα, λιγότερο flash από 12.
- **Εναλλαγή:** π.χ. **slot ~4 ώρες** ή (καλύτερα) **φάσεις γύρω από ανατολή/δύση** (NTP + κατά προσέγγιση τοποθεσία).
- **Μετάβαση:** άμεση αλλαγή εικόνας **ή** **crossfade** 30–120 s μεταξύ γειτονικών slots.
- **Lux:** μπορεί να **μετατοπίζει ελαφρά** το timing· **υστερία** 10–20 λεπτά ώστε να μην αναβοσβήνει με συννεφιά.
- **Εναλλακτικά:** μία φωτό + **overlay** opacity από ώρα/lux + προαιρετικό layer «φωτισμένα παράθυρα» τη νύχτα.
