# H.T.C. — Πλακέτα ελέγχου καυστήρα / λέβητα

Έλεγχος κινέζικου καυστήρα 5 kW (Vevor / AliExpress) για **λέβητα πετρελαίου** σε **σπίτι φίλου** (~80 m², UFH, ηλιακός 18, boiler 160 L). Το υδραυλικό σχεδιάστηκε/εκτελέστηκε από τον Romeo.

**Σχετικό project (οθόνη θερμοστάτη):** `D:\Romeos Tsakas\romeos-display-v4\` — Viewe 7″ EEZ UI· δεύτερη οθόνη στο κουτί για τον ίδιο χώρο (UI ≠ πλακέτα καυστήρα, αλλά ίδιο κτίριο).
