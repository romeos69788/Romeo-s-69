# Καυστήρας — αναφορά hardware

## Προϊόν (ακόμα στη διάθεσή σου)

| | |
|---|---|
| **Πηγή** | [AliExpress — 12V diesel air heater 5KW all-in-one](https://www.aliexpress.com/item/1005008461450039.html) |
| **Τύπος** | Κινέζικος πετρελαιοκίνητος αέρα (parking heater), «όλα σε ένα» |
| **Τάση** | **12 V** (επιλογή listing· υπάρχουν και 8KW / 24V variants) |
| **Ισχύς** | **5 kW** (επιλογή «5kw όλα σε ένα») |
| **Οθόνη listing** | Μπλε LCD + 4 κουμπιά δεξιά (↑ ↓ OK + power) |

**Σημείωση:** Στις δικές σου φωτογραφίες ο πίνακας είναι **LCDMAX V2** με **3 κουμπιά πάνω** (μεσαίο = ON/OFF). Ίδια **οικογένεια** καυστήρων· μικρές διαφορές panel ανά batch — οι **επεμβάσεις** (ON/OFF + fan R/B) παραμένουν ίδιες στη λογική.

Δεν είναι απαραίτητα brand «Vevor» — γενικός **5kW 12V diesel heater** με ψηφιακό πίνακα.

## Τι περιλαμβάνει το kit (από listing / φωτό)

- Κύριο σώμα (μαύρο πλαστικό, εισαγωγή/εξαγωγή)
- **Ψηφιακός πίνακας ελέγχου** (μπλε LCD)
- Τηλεχειριστήριο (προαιρετικό — δεν χρησιμοποιείται στο H.T.C.)
- Δεξαμενή καυσίμου, γραμμή, φίλτρο, αντλία καυσίμου
- Σωλήνες εισαγωγής/εξαγωγής, κουμπώματα

## Σημεία επέμβασης H.T.C. (επιβεβαιωμένα)

| Σημείο | Σύνδεση |
|--------|---------|
| Πίνακας LCD — **μεσαίο πάνω κουμπί** | 2 σύρματα → ρελέ → παλμός 2,5–3 s |
| Μοχλός / κύκλωμα **θερμοκρασίας** | Ψηφιακό pot (SPI) |
| **Fan** R+B από κίτρινο module | Κόψιμο → fan controller πλακέτας |
| **Δικό μας** LCD 20×4 | Παρακολούθηση Boiler / Prosagogi / Epistrofi |

## Τροφοδοσία συστήματος ελέγχου

- Καυστήρας / kit: **12 V**
- H.T.C. / πρόχειρο: buck **12 V → 5 V** (Arduino, ρελέ modules) — **ταιριάζει**
- Rev 2 ESP32: ίδια φιλοσοφία 12V in, 5V logic
