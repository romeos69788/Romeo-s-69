# HANDOFF — 2026-04-26 (ενημέρωση τέλους ημέρας)

Σκοπός: συνέχιση **άλλη μέρα** από το ίδιο σημείο — οθόνη Viewe, ROOM/SHT, Serial COM4, link μητρικής.

## Τι ισχύει τώρα (σύνοψη)

- **Serial στο CH340 (COM4):** `ARDUINO_USB_MODE=0`, `ARDUINO_USB_CDC_ON_BOOT=0` στο `boards/BOARD_VIEWE_UEDX80480070E_WB_A.json` και στα build flags του env Viewe στο `platformio.ini`. `Serial` → **UART0**, όχι native USB CDC.
- **Monitor:** `monitor_port` / `upload_port` = COM4 (προσαρμογή αν αλλάξει COM), **`monitor_rts = 0`**, **`monitor_dtr = 0`** για «σιωπηλό» COM4 με CH340.
- **ROOM I2C (προτεινόμενο):** **GPIO19 SDA / GPIO20 SCL** (ίδιο bus με GT911), flags `-DROMEOS_ROOM_I2C_SDA=19` `-DROMEOS_ROOM_I2C_SCL=20`.
- **Αισθητήρας στο firmware:** SHT4x, SHT3x, AHT20/21 (`romeos_room_sht_touch_bus_*`, `romeos_room_temp_sensor_*`). **Ξανά-ανίχνευση κάθε 45 s** αν δεν βρέθηκε στο boot.
- **Από log χρήστη:** I2C scan έδειξε **μόνο `0x5D` (GT911)** — δεν φαίνεται SHT στο bus → έλεγχος **καλωδίωσης / 3V3 / GND / SDA-SCL** (module SHT4x AliExpress είναι συμβατό, όχι θέμα «λάθους τύπου» στο software).
- **Fallback UI:** χωρίς τοπικό I2C αισθητήρα, αν **`ROMEOS_ROOM_USE_MB_FALLBACK=1`** (default) και **έγκυρη τηλεμετρία** χωρίς demo δεξιά (`mb_proto_reserved` bit0 = 0), η γωνία **ROOM** παίρνει τιμή από **`room_display_c_x10`** (μητρική). Απενεργοποίηση: `-DROMEOS_ROOM_USE_MB_FALLBACK=0` στο `platformio.ini`.

## Επόμενα βήματα (όταν συνεχίσεις)

1. **Hardware:** επιβεβαίωση συνδέσεων SHT4x → 19/20/3V3/GND μέχρι να εμφανιστεί **`ACK 0x44`** στο `[i2c0]` scan στο Serial.
2. **Build/upload:** `pio run -e BOARD_VIEWE_UEDX80480070E_WB_A -t upload` (COM4 ελεύθερο από άλλο monitor).
3. **Προαιρετικά:** διόρθωση `ROMEOS_MB_LAN_IP` στο `platformio.ini` αν άλλαξε IP μητρικής στο LAN.

## Σημείωση για παλιό κείμενο στο ίδιο αρχείο

Παλαιότερη έκδοση αυτού του HANDOFF ανέφερε **GPIO43/44** για SHT και **USB_CDC_ON_BOOT=1** — αυτό **αντικαταστάθηκε** από την παραπάνω ρύθμιση (CH340 / 19/20). Για ιστορικό δες git history ή `HANDOFF_2026-04-13.md` / `HANDOFF_2026-04-20.md`.
