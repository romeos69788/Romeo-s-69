## Γιατί V4 (όχι V3)

Το panel είναι για **ειδικό** (υδραυλικός / μηχανολόγος). Τα πλουσιότερα εικονίδια **προδιαβάζουν** το περιεχόμενο κάθε τομέα πριν το πάτημα.  
Η **V3** (λιτή, ένα εικονίδιο) παραμένει καλύτερο πρότυπο για τον **θερμοστάτη χώρου**.
