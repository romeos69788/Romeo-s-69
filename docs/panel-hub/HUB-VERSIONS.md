## Ροή (συμφωνία 2026-07-19 βράδυ)

2. Ολοκλήρωση **V2** στο EEZ  
3. PlatformIO flash V2 → COM4  
4. Σύγκριση → επιλογή (ρητό κλείδωμα μόνο όταν το πει ο χρήστης)
