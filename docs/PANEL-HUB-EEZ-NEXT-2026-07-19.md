## Διάταξη 800×480 — πρόταση (όχι lock)

Viewe = **800×480**. Έξι ισομεγέθη → φυσικά **2 σειρές × 3 στήλες**.

```
┌────────────┬────────────┬────────────┐
│   ΝΕΡΟ     │   ΑΝΤΛΙΑ   │   ΕΞΟΔΟΙ   │
├────────────┼────────────┼────────────┤
│  ΜΠΟΪΛΕΡ   │  ΣΥΣΤΗΜΑ   │ ΣΥΝΔΕΣΕΙΣ  │
└────────────┴────────────┴────────────┘
```

| Θέμα | Πρόταση agent |
|------|----------------|
| Περιθώρια | ~16–24 px γύρω |
| Κενό μεταξύ tiles | ~12–16 px |
| Tile μέγεθος (χονδρικά) | ~240×190 px το καθένα (μετά τα gaps) |
| Μέσα στο tile | εικονίδιο πάνω · ετικέτα κάτω (ή εικονίδιο+κείμενο κεντραρισμένα) |
| Header | προαιρετικό λεπτό (ώρα / «ΛΕΒΗΤΟΣΤΑΣΙΟ») — αν γεμίζει, **χωρίς** header στην v1 |
| Πάτημα | όλο το πλαίσιο clickable → `loadScreen` detail (αργότερα) |

## EEZ βήματα (πρακτικά στο PC σου)

1. Νέο screen π.χ. **`PANEL_HUB`** (800×480) — ή νέο project `romeos-panel` αν προτιμάς καθαρό ξεκίνημα
2. 6× Panel/Container (ή Button) ισομεγέθη · grid 2×3
3. Label κειμένου σε καθένα με τα ονόματα που καταλήξουμε
4. Placeholder εικονίδιο (έστω χρωματιστό τετράγωνο) μέχρι να διαλέξουμε icons
5. Export → όταν είμαστε OK οπτικά, δένουμε clicks / firmware

**Δεν** φτιάχνουμε ακόμα τις 6 detail οθόνες στο EEZ — μόνο κενά screens αν θες για navigation test αργότερα.
