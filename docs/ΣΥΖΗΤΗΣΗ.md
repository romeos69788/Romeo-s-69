## Pin map (Alpha rev A)

```
GPIO35  → CT HP (SCT-013) — μοναδικό CT
GPIO26  → Relay K1
GPIO27  → Relay K2
GPIO14  → Relay K3
GPIO12  → Relay K4 (heater)
GPIO13  → Relay K5
GPIO15  → Relay K6
GPIO4   → OneWire DS18B20 (TBD)
GPIO34  → Flow sensor
UART1   → Beta (TX=17, RX=16, 115200)
UART2   → Displays (TX=25, RX=33, 115200)
```

## MQTT (συμβατό με romeos-remote)

- **State:** `romeos/<DEVICE_ID>/state`
- **Cmd:** `romeos/<DEVICE_ID>/cmd` → `setpoint_c_x10`, `heater`
