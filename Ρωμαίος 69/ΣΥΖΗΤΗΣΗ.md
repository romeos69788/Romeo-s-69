## Αρχεία firmware που δημιουργήθηκαν

### platformio.ini

- Environments: `alpha_rev_a`, `alpha_rev_a_bench`
- Libraries: PubSubClient, ArduinoJson
- Build flags για MQTT device ID, version

### alpha_pins.h — Pin map

```
GPIO35  → CT HP (SCT-013)
GPIO26  → Relay K1
GPIO27  → Relay K2
GPIO14  → Relay K3
GPIO12  → Relay K4 (heater)
GPIO13  → Relay K5
GPIO15  → Relay K6
GPIO4   → OneWire DS18B20 (TBD)
GPIO34  → Flow sensor
UART1   → Beta (TX=17, RX=16, 115200)
UART2   → Displays (TX=25, RX=33, 115200)
```
